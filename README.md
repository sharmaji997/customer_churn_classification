# Customer Churn Classification

A machine learning project that predicts whether a bank customer is likely to leave or stay using an Artificial Neural Network (ANN). The project includes data preprocessing, feature engineering, TensorFlow/Keras model training, and a Streamlit-based web interface for real-time predictions.

## Technologies Used
- Python
- TensorFlow/Keras
- Scikit-learn
- Pandas
- NumPy
- Streamlit

## Features
- Customer churn prediction
- Interactive Streamlit GUI
- Real-time predictions
